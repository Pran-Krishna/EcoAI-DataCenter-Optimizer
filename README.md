# 🌱 GreenMind AI – Eco Server Optimizer

## 🚀 Problem
Data centers consume massive energy and contribute to carbon emissions.

## 💡 Solution
An AI-powered system that dynamically adjusts server performance based on:
- Load
- Carbon intensity
- Energy cost
- Temperature

## 🧠 Approach
We designed a reinforcement learning environment where an agent learns to balance:
- Performance ⚡
- Cost 💰
- Sustainability 🌱

## 🔥 Features
- Smart energy optimization
- Carbon saving tracking
- Adaptive decision-making
- Real-time simulation UI

## 🌍 Impact
This system can significantly reduce energy waste and carbon emissions in large-scale data centers.
