# 🌱 EcoAI – Data Center Optimizer

### Intelligent AI system optimizing performance, cost, and carbon emissions in real time

---

## 🚨 Problem
Modern data centers consume enormous amounts of electricity, resulting in:

- High operational costs 💰  
- Increased carbon emissions 🌍  
- Inefficient resource utilization ⚡  

Current systems lack adaptive intelligence to balance performance with sustainability.

---

## 💡 Solution
EcoAI is an intelligent system that dynamically manages data center operations using AI-driven decision-making.

It continuously analyzes:

- Server Load 📊  
- Carbon Intensity 🌫️  
- Energy Price 💸  
- Temperature 🌡️  

…and selects the most optimal operating mode in real time.

---

## 🧠 Core Approach
We model the system as a **Reinforcement Learning environment**, where an AI agent learns to make optimal decisions under changing conditions.

The agent balances three critical objectives:

- ⚡ Performance (efficient workload handling)  
- 💰 Cost (minimizing energy expenses)  
- 🌱 Sustainability (reducing carbon footprint)  

---

## ⚙️ System Design

### 🔹 State Space (Inputs)
- Server Load (0–1)  
- Carbon Intensity (0–1)  
- Energy Price (0–1)  
- Temperature (0–1)  

### 🔹 Actions
- 0 → Low Power Mode 🌱  
- 1 → Balanced Mode ⚖️  
- 2 → Turbo Mode ⚡  

### 🔹 Reward Logic
The agent is rewarded or penalized based on:

- Reducing carbon emissions ✅  
- Handling high workloads efficiently ✅  
- Saving energy costs during peak pricing ✅  
- Avoiding overheating ❌  
- Minimizing unnecessary switching ❌
